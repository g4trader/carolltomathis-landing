# Landing page - Mentoria de Relacionamentos | Carol Tomathis

Projeto em Next.js + Tailwind pronto para deploy na Vercel.

## Como rodar

```bash
npm install
npm run dev
```

Depois, é só conectar o repositório na Vercel e fazer o deploy.
